Project Overview

We connect three strands into a cohesive toolchain: (1) an optimised Rational Closure (RC) implementation; (2) a topic‑aware knowledge base generator that improves interpretability by mapping atoms to phrases; and (3) a web UI that explains entailments step‑by‑step in both symbolic and natural‑language form.
